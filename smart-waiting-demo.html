🚀 Smart Waiting Message - دليل الاستخدام

اختبر رسالة الانتظار الذكية بطرق مختلفة

📌 الأمثلة الأساسية

🎯 أمثلة متقدمة

📝 طريقة الاستخدام

1. عرض رسالة بسيطة:

showSmartWaiting();

2. عرض رسالة مخصصة:

showSmartWaiting({ title: 'جاري التحميل', message: 'يرجى الانتظار...', icon: '🚀' });

3. عرض رسالة مع شريط تقدم:

const waiting = showSmartWaiting({ title: 'تحميل البيانات', showProgress: true }); // تحديث التقدم waiting.updateProgress(50); // 50% waiting.updateProgress(100); // 100% // إخفاء الرسالة waiting.hide();

4. رسالة بإخفاء تلقائي:

showSmartWaiting({ title: 'تم الحفظ بنجاح', icon: '✅', autoHideAfter: 2000 // إخفاء بعد ثانيتين });

5. إخفاء الرسالة يدوياً:

hideSmartWaiting();
💡 ملاحظة: يمكنك استخدام رسالة الانتظار الذكية في أي صفحة من صفحات مشروعك. ما عليك سوى نسخ HTML و CSS و JavaScript من الصفحة الرئيسية.

⚙️ الخيارات المتاحة

الخيار الوصف القيمة الافتراضية
title عنوان الرسالة "جاري التحميل..."
message نص الرسالة "يرجى الانتظار..."
icon الأيقونة (emoji) "⏳"
showProgress إظهار شريط التقدم false
autoHideAfter إخفاء تلقائي (بالمللي ثانية) null
